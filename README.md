## Vinted Hash Toolkit

Single Page Application in React + Vite + Tailwind pensata per utilizzo mobile (iPhone) con due sezioni:

- **Smart Copy Manager**: gestione di frasi predefinite con copia veloce negli appunti e persistenza in `localStorage`.
- **Image Transformer (Anti‑pHash)**: trasforma immagini lato client tramite Canvas con leggera rotazione casuale, micro‑crop 1% e salvataggio come JPEG (qualità 0.92) senza metadati EXIF.

### Requisiti

- Node.js 18+ consigliato

### Installazione

```bash
npm install
```

### Sviluppo

```bash
npm run dev
```

Apri poi l'URL indicato in terminale (es. `http://localhost:5173`) dal tuo iPhone (stessa rete) o direttamente dal browser desktop per test veloci.
